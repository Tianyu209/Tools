#auto_convert.py
If not exist, create two folders "csv" and "parquet" in the same directory as the script.
Convert all csv files in the "csv" folder to parquet files in the "parquet" folder. Vice versa.

#gui.py
Gui has two buttons:
- File Convert: Convert a single file(csv/parquet) to the other format.
- Folder Convert: Convert all csv/parquet files in the folder to the other format. 
    - csv files store in "csv" folder. 
    - parquet files store in "parquet" folder.
